Simplified Dental Clinic Management Systems — Backup & Restore

Export all data from Google Sheets + Drive, then run locally or restore to another deployment.

Step 1 — Connect to Google Apps Script

Open the app → Settings → Sync to copy your Apps Script URL.
It looks like: https://script.google.com/macros/s/AKfycb.../exec

Step 2 — Backup

Pulls all records from Google Sheets and downloads patient photos from Google Drive. Everything is saved into a single .zip file you can store on your PC or USB drive.

Waiting for backup to start…

Step 3 — Run Locally

After extracting the backup zip, place the local-server/ folder next to the backup's data/ and uploads/ folders, then:

Install once:  cd local-server && npm install
Start server:  node server.js
Open app:  open cyrabelldental_v2.html in Chrome / Edge
Set Sync URL:  http://localhost:3001 in app Settings → Sync

The local server is a drop-in replacement for the Google Apps Script backend — all features work the same except email/SMS sending and OCR scanning (those require cloud access).

Step 4 — Restore (optional)

Upload a previously downloaded backup zip to push all data back into Google Sheets.
